Jquery Geolocation Picker (jquery-geopicker)

A quick geolocation picking jquery pluging

Download this project as a .zip file Download this project as a tar.gz file

What is this

Simply, this is a small plugin that helps you to easily create Google maps HTML control to pickup a lat/lng value and populate it to a form/control.

How to use

<input id="geolocation" />
<script>
$("#geolocation").geoPicker();
</script>

Dependencies

This plugin depends on the GMaps plugin (by Gustavo Leon), which in turn depends on jQuery and Google Maps.

Quick start

Copy and past the following lines. Modify the paths if needed (to match your project's path):

<link href="jquery-geopicker.css" rel="stylesheet">
<script type="text/javascript" src="//ajax.googleapis.com/ajax/libs/jquery/1.11.1/jquery.min.js"></script>
<script type="text/javascript" src="//maps.google.com/maps/api/js?sensor=true"></script>
<script type="text/javascript" src="gmaps.js"></script>
<script type="text/javascript" src="jquery-geopicker.min.js"></script>

Options

The plugin accepts (and passes) all the options that GMap accepts. Few more options are:

addSearch

Boolean (false), whether to add search controls to the control or not.

apiKey

String, for reversed geocoding API from Google, this key will be used. You can ignore it if you are not going to do a reversed geocoding.

autoFit

Boolean (false), if enabled, it will automatically fit the map to the added marker on initialization.

class

String, a string of classes to be added to the outer container.
